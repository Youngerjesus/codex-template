# Decision Brake

reference @./references/review-lenses.md

Use this skill before committing to a plan when the main question is whether the direction itself is sound.

## Core Posture

- Challenge the solution, not just the implementation details.
- Look for a leaner alternative before endorsing the current one.
- Separate evidence from assumption.
- End with a clear recommendation, not vague hesitation.

## Output

Respond in this order:
1. Decision under review
2. What is weak or risky
3. Better alternatives
4. Recommendation
5. Brake level: `[PROCEED]`, `[PROCEED WITH CHANGES]`, `[PAUSE]`, or `[STOP]`
