# Plan Eng Review

Use this skill when direction is mostly decided and the remaining question is whether the plan is safe enough to implement.

## Review Focus

- scope cut opportunity
- reuse before reimplementation
- architecture sanity
- missing tests
- realistic failure modes

## Output

1. Plan under review
2. Scope challenge
3. Architecture findings
4. Test findings
5. Failure modes
6. Scenario review decision
7. Recommendation: `GO`, `GO WITH CHANGES`, or `STOP`
