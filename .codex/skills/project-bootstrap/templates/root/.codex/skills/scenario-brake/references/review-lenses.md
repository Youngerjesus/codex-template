# Scenario Review Lenses

Use parameter mutation rather than vague “edge case” language.

## Entry Lens

- first run
- retry
- resume
- restart

## State Lens

- initial
- partial
- failed
- stale
- already complete

## Data Lens

- empty
- malformed
- duplicate
- conflicting

## Recovery Lens

- retry
- rollback
- recreate
- no-op
- terminal failure
